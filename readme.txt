About Project:
* Firstly you need to install json-server.
* Install using below command, json-server gets installed globally:  
	npm install -g json-server  
* Start the json-server using below command in the command prompt using products.json location.
	json-server --watch products.json --port 3000
* Here we are using json-server as our mock backend server. 
* CRUD-project is the actual angular project, you can import the project in any IDE like Visual studio code and execute (ng serve) command in terminal.
* Now you should be able to access UI in http://localhost:4200 url.
